# Hospital-API

Welcome to our cutting-edge Hospital Management API, powered by Django REST Framework! Designed with simplicity and efficiency in mind, our API caters to three essential user roles: Doctor, Patient, and Admin.

Authenticating users is seamless with Django REST Framework's custom token authentication, ensuring secure access to the system.

Below, you'll find everything you need to get started:

A hassle-free installation guide
Comprehensive API endpoints documentation
Sample requests and responses to kickstart your integration journey
Experience the power of streamlined hospital management like never before with our intuitive API solution!


![developer](https://img.shields.io/badge/Developed%20By%20%3A-Inzamamul%20Haque%20Ashique-red)

# Setup

```bash
pip install -r requirements.txt
```

**Running locally:**

```bash
python manage.py runserver
```

## Sample API Request and Response

**POST api/doctor/registration/**

Details: API endpoint for creating new doctor account.

request body:
```json
{
    "user_data": {
        "username": "doctor5",
        "first_name": "Dr. Jhon",
        "last_name": "Doe",
        "password":"doctorpass123",
        "password2":"doctorpass123"
    },
    "profile_data": {
        "department": "CL",
        "address": "Bangladesh",
        "mobile": "123123434252"
    }
}
```
response body:
```json
{
    "user_data": {
        "username": "doctor5",
        "first_name": "Dr. Jhon",
        "last_name": "Doe"
    },
    "profile_data": {
        "department": "CL",
        "address": "Bangladesh",
        "mobile": "123123434252"
    }
}
```



**POST api/doctor/login/**

Details: API endpoint for doctor login. Admin needs to approve account otherwise login will not be successful.

request body:
```json
{
    "username": "doctor7",
    "password": "doctorpass1234"
}
```
response body:
```json
{
    "token": "e617d20f74709f1d2097876696e026f602cef5b9"
}
```
